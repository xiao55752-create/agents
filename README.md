# agentOS · 本地演示模式

面向工程团队的 **Agentic OS 控制台**（纯前端）——任务说明驱动、能力包编排、修改方案人工验收、审计与价值度量。

> 默认 **无需后端**。任务、日志、验收、预算、指标、AI 平台数据均在浏览器内模拟并持久化（`localStorage`）。

## 一键启动

```bash
cd agentos
npm install
npm run dev
```

打开 **http://localhost:3000**

演示账号：`demo@agentos.local` / `agentos123`

## 六大模块

| 模块 | 能力 |
|------|------|
| **总览** | 迷你仪表盘、功能入口、5 分钟体验、工作方式 |
| **工作台** | 任务说明 → 执行 → 修改方案 → **人工验收**（通过 / 要求修改 / 打回）→ 日志与操作记录 |
| **智能体** | 模型 · 行为指令 · 能力包拓扑 · 月度用量预算 · 优化建议 |
| **AI 平台** | 数据工场（含工作台自动同步）→ 训练中心 → 模型广场 → 应用到智能体 |
| **用量监控** | 总消耗 · 按智能体预算进度 · 高消耗任务 · 与通知中心联动 |
| **架构** | 智造基地 9 步对照 · 六层架构 · 工厂闭环 · 路线图 · 实时指标 |

## 推荐体验路径

### 路径 A：5 分钟完整体验（首选 · 对内演示）

1. 总览或顶部集成栏 → **开始 5 分钟体验**（一键填充演示数据）
2. 跟随分阶段引导：**总览 → 验收 → 智能体 → AI 平台 → 用量 → 总结**
3. 在工作台 **通过** 待验收任务（验收后引导会自动进入智能体步骤）
4. 引导结束 → **查看总结**；总览 Hero 会显示 **体验进度** 徽章
5. 可随时 **继续引导** / **再走一遍** / **查看总结**

引导支持：**← →** 切换步骤 · **Esc** 跳过 · 点击进度条跳转阶段 · 聚光灯高亮当前区域。

### 路径 B：演示模式（投屏 / 客户演示）

1. 集成栏 → **演示模式**（或总览先走完 5 分钟体验）
2. 自动全屏、隐藏助手与数据操作，顶部显示演示横幅
3. **Esc** 或横幅 **退出演示** · 支持 **打印 / 导出 PDF**
4. 演示模式下仍可从横幅启动 **5 分钟体验**

### 路径 C：任务闭环（10 分钟 · 自由探索）

1. **总览** → 新建任务说明（或使用协同助手）
2. **工作台** → 等待修改方案 → 人工验收
3. **智能体** → 调整行为指令 / 能力包 / 预算
4. **用量监控** → 查看消耗与预算进度

### 路径 D：微调闭环（15 分钟）

1. 先完成路径 A 或 C，产生任务与审计数据
2. **AI 平台 → 数据工场** → 勾选「工作台任务说明 / 验收记录」→ 生成数据集 → 执行清洗
3. **训练中心** → 创建微调任务，等待完成（自动发布到模型广场）
4. **模型广场** → 「应用到智能体」→ 回智能体页保存配置

### 协同助手

右下角 **协同助手** 支持对话式引导（识别当前页面、待验收任务、预算预警），顶部 **新手引导** 可随时重温。（演示模式下隐藏）

## 演示前走查清单（约 30 分钟）

在**无痕窗口**或清空本地数据后，按顺序勾选：

- [ ] 登录 `demo@agentos.local` / `agentos123`
- [ ] 总览 **开始 5 分钟体验** → 迷你仪表盘有数据
- [ ] 工作台待验收任务 → 右侧 **通过** → 引导自动进入智能体
- [ ] 走完引导 → **体验总结** 弹窗数据正确
- [ ] 刷新页面 → 总览 **继续引导** 从上次步骤恢复
- [ ] 总览 Hero **体验进度** 徽章可打开总结
- [ ] **演示模式** 全屏投屏一轮 → Esc 退出
- [ ] `cd apps/web && npm run build` 通过

## 核心能力

- **人工验收** — 修改方案必须人工确认，不可自动完成
- **能力包最小授权** — 内置 + 自定义能力包，工具权限自动推导
- **审计轨迹** — 验收与状态变更不可变记录
- **成本流控** — 按智能体月度 Token 预算（80% 预警 / 100% 硬停）
- **价值度量** — 成功率、打回率、方案生成率、记录覆盖率
- **AI 平台演示** — 数据 → 训练 → 模型选用完整故事线

```bash
cd apps/web && npm run build
npm run preview -w @agentos/web
```

### 静态托管（Demo 推荐）

| 平台 | 配置 |
|------|------|
| **Vercel** | Root Directory 设为 `apps/web`，Build 用默认或 `npm run build` |
| **Netlify** | Base directory `apps/web`，已含 `netlify.toml` |
| **任意静态站** | 上传 `apps/web/dist/`，SPA 需回退到 `index.html` |

仓库已包含 `apps/web/vercel.json`（SPA 路由）与 GitHub Actions **Web build** 工作流（PR 自动构建）。

部署 `apps/web/dist/` 到 Vercel / Netlify / GitHub Pages 等静态托管。
